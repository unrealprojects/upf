2 Данные

T - Температура
    Посчитать
        Среднесуточное значение
        Среднемесячное значение

P,P0 - Давление
    Посчитать
        Среднесуточное значение
        Среднемесячное значение

U - Влажность
    Среднесуточная значение
    Среднемесячная значение


DD - Направление ветра
    Посчитать
        Среднесуточная мода
        Среднемесячная мода
        Среднемесячная мода по часам

Ff - Скорость ветра
     Посчитать
        Среднесуточная значение
        Среднемесячная значение
        Среднемесячное значение за час
        Максимальное значение за сутки
        Максимальное значение за месяц






Выходные данные
    Среднесуточно
        По горизонтали :: год-месяц
        По вертикали :: дни

    Среднемесячное
        По горизонтали :: месяца
        По вертикали :: года

    Среднемесячное значение за час
        По горизонтали :: год-мясяца
        По вертикале :: часы

    Среднемесячная мода по часам
        По горизонтали :: год-мясяц
        По вертикале :: часы
